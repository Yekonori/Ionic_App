# Fire Emblem Three Houses - Ionic App

## Pages List : 

- All-Characters
- Houses-Characters
- Character-Detail

## Characters Pages Details : 

### All-Characters

Display all the houses' characters.

### Houses-Characters

Display all the selected houses' characters.

### Character-Detail

Display the details of the selected character
